# Roadmap: autopunk-media-skills

## Overview

Build the most comprehensive Claude skill library for media professionals. v2.0 adds 204 atomic journalism skills in a functional taxonomy (research → writing → editing → distribution), making skills discoverable by task rather than media format.

## Phases

- [x] **Phase 1: atomic-journalism-taxonomy** - Create all 204 skill stubs + new folder structure
- [x] **Phase 2: priority-skills-stable** - Develop top-20 universal skills to stable (completed 2026-03-19)

- [ ] **Phase 4: writing-skills-stable** - Develop all writing/ skills to stable
- [ ] **Phase 5: editing-localization-stable** - Develop editing/ + translation-localization/ to stable
- [ ] **Phase 6: archive-audience-support-stable** - Develop archive-legal/, audience-distribution/, production-support/ to stable

## Phase Details

### Phase 1: atomic-journalism-taxonomy
**Goal**: Create all 204 atomic journalism skill stubs in a functional folder taxonomy (research/, pre-production/, writing/, editing/, translation-localization/, data-journalism additions, archive-legal/, audience-distribution/, production-support/) alongside existing format-based skills.
**Depends on**: Nothing
**Success Criteria** (what must be TRUE):
  1. 199 new skill stub files exist with status: planned
  2. All 204 atomic journalism tasks covered (199 new + 5 existing data-journalism skills)
  3. Existing format-based skills (tv-documentary, youtube, podcast, etc.) unchanged
  4. ROADMAP.md and CLAUDE.md updated with new taxonomy
