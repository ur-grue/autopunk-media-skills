# Claude skills for podcasters

Concept to published episode to sponsorship — without the AI-sounding filler. A free, MIT-licensed library of **394 Claude skills** for the work that actually fills a release week.

No account, no install: open a skill, copy it, paste it into [claude.ai](https://claude.ai).

## Every release week, skill by skill
- **Plan:** episode-concept-generator, guest-research-brief, interview-question-builder
- **Script:** solo-episode-script, intro-outro-writer, ad-read-script
- **Publish:** show-notes-generator, episode-summary-writer, transcript-cleanup
- **Get paid:** guest-outreach-email, sponsorship-pitch-writer, podcast-media-kit-generator

→ Full walkthrough: [the podcaster guide](../for/podcaster.md).

## Start in 60 seconds
Paste your transcript or outline into **show-notes-generator** and get publishable show notes — formatted, with chapters and links — in under a minute. Every skill is quality-tested against a seven-dimension rubric ([details](../../tests/EVAL_FRAMEWORK.md)).
