# Claude skills for YouTubers

Idea → title → script → upload → grow, with copy that doesn't read like a model wrote it. A free, MIT-licensed library of **394 Claude skills** covering the whole publishing loop, including the SEO and sponsorship parts that actually pay.

No account, no install: open a skill, copy it, paste it into [claude.ai](https://claude.ai).

## The publishing loop, skill by skill
- **Plan:** niche-video-idea-generator, seo-title-optimizer, thumbnail-concept-brief, hook-generator
- **Script:** full-script-writer, b-roll-shot-list, call-to-action-generator, shorts-script-adapter
- **Publish:** description-seo-writer, chapter-timestamps, community-post-generator
- **Grow & monetize:** content-calendar-planner, sponsorship-email-writer, youtube-media-kit-generator

→ Full walkthrough: [the YouTuber guide](../for/youtuber.md).

## Start with the hook
The first 15 seconds decide everything. Paste your topic into **hook-generator** and get three opening scripts with the reason each works. Every skill is quality-tested against a seven-dimension rubric ([details](../../tests/EVAL_FRAMEWORK.md)).
