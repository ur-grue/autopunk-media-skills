# For Documentary Producers

[← Back to README](../../README.md)

You make factual films — for broadcast, streaming, or festivals. This library covers the whole journey: from a one-line idea to a funded, scripted, delivered film. Below is a curated path through the skills most producers reach for, in the order a project actually moves.

**New here?** You don't need to install anything. Open [claude.ai](https://claude.ai), open the skill file you want, and paste its text into Claude with your own material. That's it.
