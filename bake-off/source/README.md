# Source audio

Drop the source interview here as `interview.<ext>` (e.g. `interview.wav`, `interview.mp3`, `interview.m4a`).

Also include:

- `transcript-reference.txt` — a tool-independent reference transcript if available (used as ground truth for transcript scoring; not fed to the SaaS tools).
- `LICENSE.md` — license + attribution string for downstream use (default: CC-BY 4.0, autopunk media).

This directory is git-tracked but the audio file itself may exceed GitHub's file-size limits — if so, store the file via Git LFS or link to a stable hosted copy in `LICENSE.md`. Do not commit copyrighted material.
