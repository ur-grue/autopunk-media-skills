# Receipts

Subscription invoices for spend reconciliation under [AUT-34](/AUT/issues/AUT-34) (cap $400 on this stream of [AUT-27](/AUT/issues/AUT-27) ≤ $1,000).

Save each receipt as `<tool>-<YYYY-MM-DD>-subscribe.pdf` and `<tool>-<YYYY-MM-DD>-cancel.pdf` (or `.txt` if PDF unavailable). Redact card-number details before committing.
